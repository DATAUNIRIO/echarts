# echarts

R htmlwidget for Baidu's [ECharts](http://echarts.baidu.com) 2.2.7

## Installation

You can install echarts from github with:

```R
# install.packages("devtools")
devtools::install_github("JohnCoene/echarts")
```

## Features

1. bar charts
2. line charts
3. area charts
4. pie & donut charts
5. heatmaps
6. chord diagrams
7. venn diagrams
8. gauges
9. force networks
10. word-clouds
11. maps
12. treemaps
13. candlestick charts
14. themes
15. responsive
16. animations
